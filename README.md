# Mariamo Desiigns Website

Multilingual interior designer portfolio website built with React + Vite.

## Features

- English, Georgian, Russian language switcher
- Smooth section scrolling
- Responsive desktop/mobile navigation
- Portfolio gallery modal with image navigation
- Contact form with image upload sent to `gzirishviligiorgiwork@gmail.com` via FormSubmit
- SEO basics: sitemap + robots

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run lint
npm run build
npm run preview
```

## Deploy (Vercel)

```bash
vercel --prod
```

Then keep your alias/domain pointed to the latest production deployment.
